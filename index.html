Flex Layout Examples

Flex is probably what you need 99% of the time. It's pretty much the modern standard for styling layouts of any kind of website. It also makes life so much easier when it comes to positioning elements, especially when trying to center things.

Grid Layout Examples

Grid is also becoming quite popular for more specific needs. It can create layouts similar to flex, but with more specific rules and slight complexity. It's also a lot easier to just use flex, however when flex fails in more specific situations, grid fills the gap.

red

green

blue


.parent {
  display: grid;
  grid-template-rows: 1fr 1fr 1fr;
}

red

green

blue


.parent {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}

Hover (:hover)

Hover is a super common pseudo class. This class lets you change the CSS of an element when the user hovers over it. A common use for this is when hovering over a button, it would slightly change in color and size.

Button 

.element {
    color: #000;
    background-color: #fff;
    border: 1px solid #000;
}

.element:hover {
    color: #fff;
    background-color: #000;
}
